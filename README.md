# Mood Music 🎧 (EM EXECUÇÃO)

Projeto que recomenda músicas com base na emoção do usuário.

## 🚀 Tecnologias
- Node.js
- MySQL
- Express

## 💡 Funcionalidades
- Buscar músicas por emoção
- API REST
- Integração com banco de dados

## 🔜 Próximos passos
- Front-end em React
- Player de música (embed)
- Interface dinâmica por emoção
